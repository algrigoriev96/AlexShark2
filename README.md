Сайт рабочий, запускается через Developer Powershell (Ctrl+ё)
Указываем путь до myproject (Пример: cd C:\Users\****\Documents\AlexShark\myproject). 
Также создает среду для запуска : .\venv\Scripts\activate
После этого вводим: python manage.py runserver
Открываем в браузере http://127.0.0.1:8000/
